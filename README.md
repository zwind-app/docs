# Zwind

## 如何使用 Zwind 实现夸克搜索转存 + 追更

<a href="https://apps.apple.com/app/id6755239096" target="_blank">Zwind</a> 创造了独特的“以目录名触发搜索转存”的丝滑体验方式，实现懒人追剧，也再不用费心整理资源～

配置好夸克账号后，你只需要在解析器绑定的目录下创建一个空目录，并将其命名为你想找的资源名称，它就会自动去搜索并转存相关资源到这个文件夹。并且还会自动追更转存。

### 准备夸克账号

准备夸克账号，获取夸克 Cookie（可以使用<a href="https://quark-tool.zwind.app/" target="_blank">小工具</a>)）。


### 配置解析器

在 Zwind 中创建 WebDAV 服务器，并配置解析器。

1. 打开 Zwind，点击左下角“加号”按钮，选择“创建新服务器”。存储类型选择“夸克云盘”，粘贴“夸克 Cookie”。
2. 在“解析器绑定 - 可用解析器”处点击添加，解析器选择“夸克资源自动转存解析器”，点击保存。
3. 推荐打开“高级 - 应用启动时自动启动”开关。
4. 点击保存，创建该 WebDAV 服务器。

### 进行搜索转存

1. 回到主界面，选中新创建的 WebDAV 服务器，点击右下角启动按钮，启动 WebDAV 服务器。此时会进入软件自带的 WebDAV 浏览器，可以看到出现夸克云盘里的目录内容。
2. 点击进入绑定了解析器的文件夹（默认为 “Imports”），点击右下角加号按钮选择“创建新文件夹”。
3. 在弹出的文件夹命名输入框中，将默认的文件夹名称删除，输入自己想搜索的资源名称。点击“创建”。
4. 然后等待几秒，在目录中下拉刷新，可看到刚才添加的文件夹。点击进入。
5. 在刚才创建的文件夹中，等待几秒，拉下刷新，可以看到资源被自动转存进该文件夹中。

### 追更操作

1. 自动追更：打开 Zwind，启动对应 WebDAV 服务器，会进行自动追更。资源更新后会有提示。
2. 手动追更：在资源目录中，点击右下角加号按钮，点击“更新转存”触发手动追更。
